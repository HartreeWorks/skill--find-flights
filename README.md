# Find flights

A Claude Code skill for searching flights across SerpApi (Google Flights) and Skyscanner, ranking options by price, routing, baggage, seats, and time trade-offs, and optionally adding a booked flight to Google Calendar.

## Documentation

See [SKILL.md](./SKILL.md) for complete documentation and usage instructions.

## Installation

```bash
# Run install
npx skills add HartreeWorks/skill--find-flights

# When asked "Which agents do you want to install to?", select "Claude Code"
# in addition to the default "Universal" list.
```

If you get "command not found", [install Node](https://github.com/HartreeWorks/skills/blob/main/how-to-install-node.md) then try again.
